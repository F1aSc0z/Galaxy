# 🚀 ГАЛАКТИКА — Космический шутер

Браузерная игра в жанре **вертикальный шутер**, написанная на React + TypeScript с использованием Canvas API.

## 🎮 Демо

Запусти локально: `npm start` → [http://localhost:3000](http://localhost:3000)

---

## 📸 Скриншоты

> Меню с таблицей рекордов, игровой процесс и экран окончания игры

---

## ⚙️ Технологии

| Технология | Версия |
|---|---|
| React | 18.2 |
| TypeScript | 4.4 |
| Canvas API | — |
| Web Audio API | — |

Сторонние библиотеки не используются — только стандартные браузерные API.

---

## ✨ Возможности

- **3 типа врагов** с разными характеристиками:
  - 🔴 Basic — 3 HP, средняя скорость
  - 🟠 Fast — 2 HP, высокая скорость
  - 🟣 Tank — 6 HP, медленный, много очков
- **Система уровней** — сложность растёт по мере набора очков
- **Компаньоны** — появляются на уровне 2 (1 шт.) и уровне 5 (2 шт.)
- **Power-up'ы** — щит и дополнительная жизнь выпадают в ходе игры
- **Звуковые эффекты** через Web Audio API (без сторонних файлов)
- **Таблица рекордов** — топ-5 результатов хранится в localStorage
- **Режим огня** — авто или ручной (настраивается в меню)
- **Поддержка телефона** — сенсорное управление, адаптивная вёрстка

## 🕹️ Управление

| Действие | Клавиатура | Мышь / Тач |
|---|---|---|
| Движение | WASD / ←↑→↓ | Мышь / палец по экрану |
| Выстрел (ручной режим) | Пробел | Клик / касание |
| Выстрел (авто режим) | — | автоматически |

## 🚀 Запуск проекта

### Требования
- Node.js 16+
- npm

### Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/F1aSc0z/Galaxy.git
cd Galaxy

# Установить зависимости
npm install

# Запустить в режиме разработки
npm start
```

Откроется браузер по адресу [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена

```bash
npm run build
```

Готовые файлы появятся в папке `build/`.

## 🎯 Игровая механика

- Очки начисляются за уничтожение врагов: Basic +20, Fast +30, Tank +80
- Уровень повышается каждые 300 очков
- С ростом уровня враги появляются быстрее и двигаются шустрее
- Щит поглощает урон; при потере всех 3 жизней — конец игры
